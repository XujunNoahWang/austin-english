## 使用说明

- 当用户点击 **O/o** 按钮时，会播放 `o.MP3`（字母名称）
- 当用户点击 **/ɒ/** 按钮时，会播放 `o_a_sound.MP3`（音素发音）
- 当用户点击 **/oʊ/** 按钮时，会使用语音合成播放（因为 o_long.MP3 待添加）

## 音频格式要求

- 格式：MP3
- 建议时长：1-3秒
- 建议音质：清晰、无背景噪音
- 建议音量：适中，不要过大或过小

## 文件命名

请严格按照上述文件名命名，系统会根据这些文件名来加载音频。
