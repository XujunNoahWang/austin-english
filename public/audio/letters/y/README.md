# 字母 Y 音频文件说明

请在此文件夹中放置以下音频文件：

## 需要的音频文件

1. **y.MP3** - 字母名称 "Y" 的发音（读作 "why"） ✅ 已添加
2. **y_sound.MP3** - 音素 /j/ 的发音（如 "yes" 中的 y 音） ✅ 已添加

## 使用说明

- 当用户点击 **Y/y** 按钮时，会播放 `y.MP3`（字母名称）
- 当用户点击 **/j/** 按钮时，会播放 `y_sound.MP3`（音素发音）

## 音频格式要求

- 格式：MP3
- 建议时长：1-3秒
- 建议音质：清晰、无背景噪音
- 建议音量：适中，不要过大或过小

## 文件命名

请严格按照上述文件名命名，系统会根据这些文件名来加载音频。
